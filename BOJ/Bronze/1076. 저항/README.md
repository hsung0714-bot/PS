# Bronze2 저항

## 문제 요약
저항의 색깔 3개가 주어지며, 첫 번째와 두 번째 색은 숫자를 나타내고 세 번째 색은 곱을 의미한다. 
이를 통해 저항값을 계산하는 문제이다.

## 입력
첫째 줄에 첫 번째 색  
둘째 줄에 두 번째 색  
셋째 줄에 세 번째 색  

색은 다음과 같다.  
black, brown, red, orange, yellow, green, blue, violet, grey, white  

## 출력
저항의 값을 출력한다.

## 접근 방법
초반에는 세 번째 입력값 처리 때문에 많은 고민을 했다.
하지만 해당 값의 크기만큼 10을 거듭제곱하면 된다는 사실을 알아내고 빠르게 풀이 방법을 고민했다.
가장 고민했던 부분은 문자열을 어떻게 숫자형(long)으로 변환할 것인가였다.
이를 처리하기 위해 int main() 함수 하나만으로 해결하기 어렵다고 생각해 
외부 함수를 따로 정의하여 문제를 해결해야겠다 생각했다.

## 핵심 포인트
- 문자열을 숫자로 바꾸는 외부함수 정의가 중요.
- long 타입 사용 (값이 커질 수 있음)
- 10의 거듭제곱을 사용

## 실패 기록
- 없음

## 배운점
- 규칙을 파악하면서 문제를 단순화할 수 있었음
- 문자열 → 숫자 변환은 별도의 함수로 분리하면 코드가 훨씬 깔끔해진다는 것을 느낌
- 문제를 나눠서 생각하는 것이 구현을 쉽게 만들 수 있음
