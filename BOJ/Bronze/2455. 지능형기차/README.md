# Bronze3 지능형기차

---

## 문제요약

기차가 총 4개의 역을 지나면서, 각 역마다
- 내린 사람 수
- 탄 사람 수
가 주어진다.

각 역을 지날 때마다 기차에 탑승 중인 사람 수를 계산하면서,
기차 안에 있었던 최대 인원 수를 구하는 문제.

---

### 입력
각 역에서 내린 사람 수와 탄 사람 수가 빈칸을 사이에 두고 첫째 줄부터 넷째 줄까지 역 순서대로 한 줄에 하나씩 주어진다. 

### 출력
첫째 줄에 최대 사람 수를 출력한다.  

---

## 접근 방법
솔직히 처음에는 변수를 와다다 쓰면 쉽게 풀릴거라 생각했지만 그게 정답이 아닐거라고 생각하고 있었기에 다른 방법을 고민함.
고민하다보니 for 문을 사용해서 각 상황마다 수가 큰지 아닌지를 구분해서 가장 클때를 저장하고 그걸 보여주면 되겠다고 생각함.

---

## 핵심 포인트
- 매 역마다 최댓값 비교 후 대입

없음
