# Bronze3 최댓값

---

## 문제요약

- 9×9 크기의 배열이 주어짐
이 중 최댓값을 찾고,
그 값의 행과 열 위치(1부터 시작)를 출력하는 문제

---

### 입력
첫째 줄부터 아홉 번째 줄까지 한 줄에 아홉 개씩 수가 주어진다. 주어지는 수는 100보다 작은 자연수 또는 0이다.

### 출력
첫째 줄에 최댓값을 출력하고, 둘째 줄에 최댓값이 위치한 행 번호와 열 번호를 빈칸을 사이에 두고 차례로 출력한다. 최댓값이 두 개 이상인 경우 그 중 한 곳의 위치를 출력한다. 

---

## 접근 방법
어쩌면 처음 고민했을때 더 좋은 풀이 방식으로 생각을 했었다.(굳이 배열 저장 안 하고 입력 받으면서 처리)
근데 이 방법이 문제 의도와는 맞지 않다 생각해서 다시 배열로 푸는 방식을 고민했고 이중 for문을 사용하는 방식으로 진행했다.
---

## 핵심 포인트
- 이중 for문 사용
- 좌표 출력 시 +1 반드시 해줘야 함

## 실패 기록
없음

## 배운점
- 좌표 출력을 할 때 +1 을 까먹지 마라..
- 내가 쓴 풀이가 항상 최고의 것은 아니다.(입력 받으면서 처리가 더 좋은 풀이)
