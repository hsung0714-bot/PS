# Bronze2 소수찾기

## 문제 요약
주어진 N개의 자연수 중에서 소수의 개수를 구하는 문제이다.

## 입력
- 첫째 줄에 자연수의 개수 N이 주어진다.
- 둘째 줄에 N개의 자연수가 공백으로 구분되어 주어진다.

## 출력
- 주어진 수들 중 소수의 개수를 출력한다.

## 접근 방법
소수판별을 하는 방법은 유명해서 익히 알고 있엇다.
외부함수를 사용하는 점, √n까지만 사용해서 소수판별은 진행하는 점 등을 복습한다 생각하고
문제를 풀이했다.

## 핵심 포인트
- 시간 최적화를 위해 √n까지만 검사
- 외부 함수를 사용해 소수 판별

## 실패 기록
- 없음

## 배운점
- 소수 판별에서 √n까지만 확인해도 충분하다는 점

## 분류
- 수학
- 정수론
- 소수 판정
